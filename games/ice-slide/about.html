Ice Slide

A grid-based puzzle game where a penguin slides on frictionless ice. Each move sends the penguin gliding in that direction until it collides with a wall or the grid boundary — it cannot stop partway. Guide the penguin to the glowing star in 15 handcrafted puzzles of increasing complexity.

How to Play

  • Use Arrow Keys or swipe on the canvas to move the penguin
  • The on-screen directional buttons also work on touch devices
  • The penguin slides until it hits a wall — it cannot stop in open ice
  • Navigate the penguin to reach the ⭐ star to complete the level
  • Plan your path using walls as stopping points to redirect the penguin
  • Completing each puzzle unlocks the next level automatically
  • The move counter tracks how many slides you used — try to minimize it

About This Game

Ice Slide belongs to the family of sliding-block puzzles. The core mechanic — a piece that cannot stop until it hits an obstacle — transforms ordinary grid navigation into a spatial reasoning challenge. On the first few levels the solution path is obvious: one or two slides and the penguin arrives at the star. As levels progress, indirect routes become necessary. You must use walls as stepping stones, sliding the penguin into a wall to redirect it toward the goal, sometimes requiring four or more moves to arrive from a specific angle.

The puzzle design escalates gradually. Early levels introduce the sliding mechanic with open grids where direct paths exist. Mid-game levels add more walls in more complex arrangements, requiring you to plan sequences rather than single moves. Late levels feature tightly packed walls where the correct path requires using nearly every wall in the grid in sequence, and where an early wrong turn can strand the penguin in a dead-end with no path to the goal.

The move counter adds a secondary challenge for completionists: completing a level is satisfying, but completing it in the minimum possible moves requires solving not just the path but the optimal path. The minimum solution for each level is built into the design, so a perfect run always exists — finding it is the deeper challenge.

Why We Built It

Sliding-puzzle mechanics appear in many classic games, from the Pokémon ice caves to the sliding block puzzles of the Game Boy era. We implemented Ice Slide as a clean, distraction-free version of the mechanic with 15 purpose-built levels that ramp difficulty smoothly rather than spiking at random. The puzzle-first design means every level has a clear solution — there are no impossible states, no randomness, and no time pressure. It is a game about thinking, not reflexes, making it a good counterbalance to the faster-paced action titles in the arcade.

Tips & Strategy

  • Work backwards from the star — identify which direction the penguin needs to approach the goal from, then figure out what wall enables that final slide.
  • Map the walls — before moving, trace where each possible slide would take the penguin to build a mental map of reachable positions.
  • Don't slide into corners — trapped in a corner with no path back to useful walls means restarting the level.
  • Two-step setups — many solutions involve moving into a specific wall so the next slide lands in exactly the right position.
  • Restart freely — the Restart button resets only the current level with no penalty; use it whenever a path feels stuck.

Ideas for Improvement

  • Par score — show the minimum number of moves required per level as a par, encouraging optimal solutions.
  • Level editor — a drag-and-drop tool for creating and sharing custom ice puzzles.
  • Moving obstacles — walls that shift position each turn, adding dynamic elements to the static grid.
  • Multiple targets — puzzles requiring the penguin to visit two or more stars in sequence before finishing.
