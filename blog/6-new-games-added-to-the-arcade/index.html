6 New Games Added to the Arcade

We just shipped six new games, ranging from quick reflex tests to slower-paced logic puzzles. They're all live now and free to play, no install required. Here's a rundown of what's new.

Neon Dodge

Use the arrow keys to dodge incoming red blocks and survive as long as possible. Simple to learn, hard to master once the pace picks up.

Play Neon Dodge →

Memory Grid

Watch the tiles light up in sequence, then tap them back in the same order. Each round adds one more step to remember.

Play Memory Grid →

Lane Rush

Switch lanes to dodge oncoming obstacles and rack up points the longer you survive. A racing-style reflex test at heart.

Play Lane Rush →

Block Stacker

Stack falling blocks, clear full lines, and keep the stack from reaching the top. A classic puzzle loop, rebuilt for the browser.

Play Block Stacker →

Trap Scout

Use number hints to figure out where the traps are hiding, then clear every safe tile without setting one off.

Play Trap Scout →

Sudoku

Fill in the blanks so every row, column, and box checks out. A no-frills version of the puzzle classic.

Play Sudoku →
